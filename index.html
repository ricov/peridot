Peridot is an HTML & CSS template for quickly building web applications. The design is largely influenced by the YUI App Theme, but the underlying code is built upon the 960 Grid System. The CSS is compiled using SASS, which makes creating new themes very simple.

Download

You can download this project in either zip or tar formats. You can also clone the project with Git by running: 

$ git clone git://github.com/zroger/peridot
